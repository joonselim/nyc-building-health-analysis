NYC Building Management Health Score

Each building receives a Building Health Score (0–100) based on NYC HPD (Housing Preservation and Development) public violation records. A lower score means more unresolved violations, slower resolution, and more hazardous issues — signals that a building's current management company is underperforming. Across 19,170 buildings analyzed, there are currently 81,137 open violations tracked.

How the Score is Calculated

Weighted composite of five components, each normalized 0–100 across the full dataset. Higher component score = better management on that dimension.

30%
Open Violation Count
Total unresolved violations. More = lower score.
25%
Violation Severity
Class C × 3, Class B × 2, Class A × 1. Weighted sum.
20%
Resolution Speed
Avg days to close a violation. Slower = lower score.
15%
Recency
Violations issued in the last 6 months. More recent = lower score.
10%
Repeat Violation Rate
Same violation type appearing multiple times = lower score.

Score Distribution

Labels use percentile thresholds (bottom 10% = Poorly managed, 10–25% = Struggling, 25–50% = Moderate, top 50% = Well-managed) because a handful of severe outlier buildings would otherwise compress the scale.
