Daisy Prospect Intelligence 04
Building health score

A weighted composite, 0–100.

Each component is normalized across all buildings, then combined with explicit weights. Lower score = worse management = hotter prospect.

30%
Open violations
Total unresolved. More = lower score.
25%
Severity
Class C ×3, B ×2, A ×1. Weighted sum.
20%
Resolution speed
Avg days to close. Slower = lower score.
15%
Recency
Violations in last 6 months.
10%
Repeat rate
Same violation type recurring.
score = 0.30·open + 0.25·severity + 0.20·resolution + 0.15·recency + 0.10·repeat
